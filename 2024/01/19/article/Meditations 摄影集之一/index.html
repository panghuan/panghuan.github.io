Meditations 摄影集之一

日期： 2024-01-19     分类： 文集

简介

这是为网站摄影集制作的图片墙展示网页，第一版的模板实现了基本的展示效果。

页面截图

成果复盘

在这一版的开发中，完成的主要的几点工作：

  1. 使用 hexo 来作为静态页面生成工具，从零开始掌握其主题模板的配置，完成画廊页、图片页的制作。
  2. 掌握 tailwindcss 框架，并利用这个 CSS 框架实现响应式布局。
  3. 引入 fancyapp 这个组件来作为图片幻灯展示效果，并适应移动端操作。
  4. 将页面部署到 GitHub Pages，并以静态博客的形式发布。
  5. 将静态资源部署到阿里云 OSS 香港节点，并利用其图片处理功能。
  6. 购买域名，并将国内访问解析到阿里云，国外访问解析到 GitHub Pages。
