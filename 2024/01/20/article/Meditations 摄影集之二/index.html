Meditations 摄影集之二

日期： 2024-01-20     分类： 文集

简介

这是为网站摄影集制作的图片墙展示网页，第二版的模板实现了不同的主题效果。

页面截图

成果复盘

在这一版的开发中，完成的主要的几点工作：

  1. 使用 daisyui 来配合 tailwindcss ，利用其主题功能和部分组件模块，并通过 theme-change 来切换主题。
  2. 参考站酷的一个页面模板Tink 影像故事，设计第二版的页面。
  3. 使用 AOS 来实现 CSS3 页面滚动可见动画效果。
  4. 由于 Github 作为静态资源存储不具备图片处理功能，所以改为部署 CDN 。对于国内访问不变，国外访问走 Fastly 的 CDN 网络，再回源到阿里云香港 OSS 。
