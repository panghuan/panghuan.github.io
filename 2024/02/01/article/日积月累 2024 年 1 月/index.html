日积月累 2024 年 1 月

日期： 2024-02-01     分类： 文集

这里是对每月各事项进度所做的一个回顾，记录在阅读、生活等方面的思考和收获。这是写给自己的，在未来的某一天回顾，这些时光如同雁过留痕，能看到些许印记。

to-do-list 进度表

鉴于 deadline 是第一生产力，因此考虑使用进度表的方式，来直观掌握各个事项的完成情况，本质上是一种自我督促。

书籍阅读

这个月读完了加缪的长篇小说《鼠疫》。这本书多年前就进入了待读清单，之前加缪的《局外人》曾经也给我留下了深刻印象，优秀的作家总是有着超乎寻常的洞察力。

这本书我的豆瓣评语是：「《鼠疫》这本书好就好在，“鼠疫”不仅仅代表着鼠疫。」

这本书里，自然也有很多金句广为流传，随便摘抄一段感受一下。

过分抬高义举，最终会间接地大力颂扬罪恶。因为，这会让人猜想，义举十分罕见，才显得如此可贵，而邪恶与冷漠则是人的行为更常见的动力。

……

世间的罪恶，几乎总是来自愚昧无知，善意如不明智，就可能跟邪恶造成同样的损害。

——《鼠疫》

思考笔记

这个月开始将日常折腾的收获记录到卡片笔记中，比如寻找某个解决方案或者工作流程，尝试了很多个产品或方法，挺花时间的。可能一段时间或若干年后，又会遇到这个需求要解决，之前的成果没有记录的话就会忘记，于是又要折腾一遍。虽然说折腾是一种乐趣，但是就结果而言还是需要更高效些。这次把尝试过程中，各种选择的优缺点及时记录下来，对于最终选择和注意事项也写明白，通过可以搜索的方式归档。

只有被索引的知识，才存在被吸收内化的可能。

另外，老唐再次提到了对于经济学的看法，跟自己的思考契合，遂一并记录下来，不再反复造轮子。

技术收获

这个月痛下决心，把较多的时间投入到博客的各项工作上，算是完成了阅读-思考-记录-输出成型的完整闭环。建立一个顺畅的工作流程非常关键，这是形成良好习惯的基础。

  1. 使用 hexo 和 tailwindcss 框架来制作静态页面，利用 hexo 主题自带的模板发布文章，用之前设计好的页面模板收集游戏摄影截图，以及新开发一个页面实现菜品信息的查询。
  2. 搞定了域名、对象存储和 CDN，实现国内外分流和云端图片处理，优化了代码流程。将静态博客发布到域名节点，并用 GitHub Actions 实现一键部署静态资源。
  3. 文字工具采用支持 Markdown 语法所见即所得的编辑器 Typora ，并配置好图床和写稿时图片插入上传。文章中配图的处理交给本地应用 PhotoScape X 。页面评论采用 Twikoo 的支持。
  4. 为了实现菜品信息查询，花了比较多时间学习 algolia 搜索工具的集成。虽然说大体满足了需求，但还是感叹过程中阅读技术文档的熟练度还不够。

总得来说，能够赶在新春之前，完成页面的上线，算是基本完成了预定目标。但这些技术折腾都只是形式，真正重要的，还是日积月累将所思所想不断提炼、良好表达展现，为日后回顾复盘提供宝贵的资料。

美食烹饪

这个月尝试的新菜式远远超出了预料，其实只要愿意动手，不怕麻烦，就会有收获，当然烹饪的熟练度和菜式还有很多提升空间。
